# CityForge
**All-in-one cheat panel for Cities: Skylines 2**

Money, demand, construction, buildings, storage and more, all in one place, no restarts needed.

> ⚠️ **This is my first mod and still actively in development. Bugs and issues may occur, feedback is always welcome!**

---

## Features
- 💰 Money editor
- 📈 Demand control
- 🏗️ Construction tools
- 🏢 Building management
- 📦 Storage controls
- ⚡ All changes apply instantly, no restarts required

## Installation
1. Subscribe on [Paradox Mods](https://mods.paradoxplaza.com)
2. Enable the mod in the in-game mod manager
3. Launch Cities: Skylines 2
